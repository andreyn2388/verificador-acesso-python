# Verificador de Acesso em Python

Este é um projeto simples que desenvolvi enquanto estou aprendendo Python.

A ideia do projeto é treinar a lógica de programação utilizando apenas conceitos básicos, como `input`, `if` e `else`.

## O que o programa faz
O programa pergunta:
- o nome da pessoa
- a idade
- se ela possui documento

Com base nessas informações, ele informa se a pessoa pode ou não ter acesso.

##  Objetivo do projeto
Meu objetivo com esse projeto foi praticar:
- tomada de decisão com if e else
- comparação de valores
- entrada de dados pelo usuário

Esse projeto faz parte do meu processo de aprendizado em programação e lógica.
